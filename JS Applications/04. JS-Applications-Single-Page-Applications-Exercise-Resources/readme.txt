Поради използването на различни сървъри (http-server / lite-server / live-server) 
и техните имплементации - може да възникне проблем с импортирането на файлове
 (CSS, HTML или други скриптове). Ако нещо не работи - коригирайте пътя.

Задачата за Филмите е решена с innerHTML при създаването на карта на филама(Details) за разлика от лектора Виктор.
Tова води до по-различен подход към извикването на функциите за бутоните: Edit,Delete,Like.
